On the Loss of Injectant and Apparent Dispersivity in Single-Well Push-Pull Tests Influenced by Natural Drift

Authors: Sverre Gullikstad Johnsen, Curtis H. Whitson

Venue: Transport in Porous Media (2010), Open Access

DOI: 10.1007/s11242-010-9632-6

Citations: not catalogued

Push-Pull Test Apparent Dispersivity Critical Time Monte-Carlo Correlation Pickens-Grisak Closed-Form Prediction Phase 3

Context

A short follow-up to Johnsen-Whitson 2008 TIPM that derives three new analytical results from the 2008 streamline model: a closed-form expression for the critical injection-phase duration beyond which injectant is lost to natural drift, a Monte-Carlo regression yielding a log-log linear correlation for single-layer best-fit Peclet number (and hence apparent dispersivity) as a function of the test-design ratios, and a layer-by-layer Muskat 1949 summation that predicts the Pickens-Grisak SW1/SW2 stratified-aquifer production profiles from the single-layer correlation. The paper closes the 2008 framework by replacing its full streamline integration with a closed-form correlation accurate to within ±30% in the Peclet number - which translates into a much smaller uncertainty in field-data-matching as the Pickens-Grisak prediction shows. Sverre Johnsen's NTNU PhD work with Curtis as co-supervisor.

Key Contributions

  • Closed-form critical injection-phase duration Tc. The time at which the first injectant particle crosses the production-phase stagnation point (located at (x, y) = (-Qp/Uu, 0) in the drift-aligned frame) and becomes unrecoverable:

    Tc = -[Qp + Uu rw - Qi ln|(Uu rw + Qi) / (Qi - Qp)|] / Uu2

    with Qi the areal injection rate (volumetric divided by height), Qp the areal production rate, Uu the drift velocity, and rw the well radius. If injection duration Ti > Tc, some injectant is lost and the effluent profile develops pronounced tailing as Ti → Tc. For a stratified reservoir, Tc varies layer by layer, so conservative production in some strata can coexist with injectant loss in others. Tc closes the previously-implicit recoverability assumption of the 2008 model and gives a pre-test design check.
  • Single-layer apparent-dispersivity correlation from Monte-Carlo regression. Running the 2008 streamline model for 1,000 stream-lines on each of 1,000 independently-sampled parameter sets (Qi, Qp, Uu log-uniform around the Pickens-Grisak SW1 values; Ti log-uniform in [0.01 Tc, Tc]), fitting the 1-D Aronofsky-Heller convection-dispersion solution to each simulated profile gives Peclet numbers that collapse onto a single log-log line:

    NPe ≈ 7.5 (Ti/Tc)-1 (Qi/Qp)0.6

    Converting via αa = L/NPe with average travel length L = 2(√(2 Qi Ti + rw2) - rw) gives the single-layer apparent dispersivity due to natural drift alone:

    αa ≈ 0.13 L (Ti/Tc) (Qi/Qp)-0.6

    Plots linearly versus travel length on a log-log scale, reproducing the empirical compilations of Arya 1988 and Mahadevan 2002 without invoking any physical dispersion. The ±30% uncertainty on the prefactor reflects irreducible scatter across the parameter space and is shown to be insignificant for predicting the Pickens-Grisak effluent profiles.
  • Stratified push-pull production-profile prediction via Muskat 1949 weighting. For a stratified reservoir with no cross-flow, overall producing concentration is the permeability-thickness-weighted Muskat combination of single-layer profiles:

    C = ∑j(Cj bj kj) / (b kA)

    where bj, kj are layer thickness and permeability and kA is arithmetic-average permeability. Workflow: compute Tc,j per layer from the closed-form expression; compute NPe,j from the correlation; compute single-layer profiles from Aronofsky-Heller; combine via Muskat. Applied to Pickens-Grisak SW1 and SW2: predicted profiles match field data across the full Vprod/Vinj range with ±30% bracket curves within field-data scatter. Uses only test parameters, layer conductivities, and drift velocity - no fitting.

Significance and Contribution

The 2008 framework operationalised. The 2008 Johnsen-Whitson paper had argued that macro-scale apparent dispersivities in single-well push-pull tests are a conformance fingerprint of drift plus stratification, not a scale-dependent rock property - but the demonstration required running the full streamline model. The 2010 follow-up replaces the streamline integration with a closed-form correlation (Eq. 3) plus the Muskat summation (Eq. 5), giving practitioners a routine tool to interrogate any echo-test apparent dispersivity. Plug in known test parameters and layer properties; compare the predicted central profile to the field data; any residual is a principled upper bound on physical dispersion and un-modelled conformance mechanisms.

The drift-budget tool. The single-layer correlation gives a first-cut estimate of how much of a reported field-scale apparent dispersivity is plausibly drift-induced, using only test design parameters and site drift velocity. Deviations above the correlation are candidates for additional conformance mechanisms (stratification, partial-completion effects) rather than physical mixing.
